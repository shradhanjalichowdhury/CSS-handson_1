What is CSS?

  1. CSS stands for Cascading Style Sheets.
  2. CSS describes how HTML elements are to be displayed on screen,paper, or in other media.
  3. CSS is not neccesaryCSS saves a lot of work. It can control the layout of multiple web pages all at once
  4. External stylesheets are stored in CSS files.

CSS can be added to HTML documents in 3 ways:

  • Inline - by using style attribute inside HTML elements1
  • Internal - by using a style element in the head section
  • External - by using a link element to link to an external CSS file
  • The most common way to add CSS, is to keep the styles in external CSS file.However, in this tutorial we will use inline and internal styles, because it is easier the demonstrate, and easier for you to try to yourself.

When comparing CSS class vs ID, the difference is that CSS class applies a style to multiple elements.ID, on the other hand, applies a styleto one unique elements. ID is also speial in that you can use a special URL to link directly to an element and it's used by JavaScript.


In CSS, selectors are used target a specific element or range of elements on a web page. Once a element has been targeted, a style or set of styles can be applied to an element.


There is a wide range of selectors available.Two of the most commonly used are class and ID. Both are used to target elements to which a style should be applied.
